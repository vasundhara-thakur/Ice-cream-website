#  Ice Cream Website

A responsive and interactive Ice Cream Website built using **HTML, CSS, and JavaScript**.  
This project focuses on smooth UI interactions, dynamic content updates, and modern frontend practices.

---

##  Features

-  Dynamic background color change based on selected flavor
-  Ice cream image changes with smooth pop-from-bottom animation
-  Fully responsive design
-  Animated hamburger menu
-  Smooth navigation with Home reset functionality
-  Clean and scalable JavaScript data-driven approach

---

##  Built With

- **HTML5** – Semantic structure  
- **CSS3** – Flexbox, animations, transitions, media queries  
- **JavaScript (ES6)** – DOM manipulation, event handling
